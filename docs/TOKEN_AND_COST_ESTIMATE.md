# 토큰·비용 추정 (1번: 한국 유지 + 영어/해외 실험)

## 사용 모델

- `.env` 기준: **`gemini/gemini-2.5-flash`**
- Flash 계열이라 상대적으로 저렴한 편.

---

## Crew 1회 실행 시 토큰 추정

- **구성**: 7개 태스크 순차 실행(리드 조사 → 감사 → 랜딩 → 마케팅 → 제안서 → 아웃리치 → Notion 로깅).
- **특징**: 뒤로 갈수록 이전 태스크 출력이 context로 붙어서 **입력 토큰이 누적**됨.

| 구간 | 입력(대략) | 출력(대략) | 비고 |
|------|------------|------------|------|
| 1. lead_research | ~3k | ~6k | 10개사 테이블 |
| 2. website_audit | ~10k | ~8k | 감사 테이블 |
| 3. landing_page | ~20k | ~15k | 회사별 랜딩 초안 |
| 4. marketing_recommendation | ~38k | ~6k | 채널 추천 |
| 5. proposal | ~47k | ~25k | 제안서(요약+상세+사례) |
| 6. email_outreach | ~75k | ~18k | 4-touch 시퀀스 |
| 7. notion_logging | ~95k | ~8k | 로그 요약 + Notion 호출 |
| **합계** | **~290k** | **~86k** | 회사 10개 기준 |

- **영어로 돌려도** 비슷하거나 약간 많을 수 있음(단어 수 증가). **대략 300k in / 90k out** 으로 잡으면 여유 있음.

---

## 비용 감당 가능한지 (대략)

- Google AI 스튜디오/API **Gemini 2.5 Flash** 구간 예시 (공식 요금제에 따라 다름):
  - Input: 약 **$0.15 ~ $0.30 / 1M tokens**
  - Output: 약 **$0.60 ~ $2.50 / 1M tokens**
- **1회 런** (300k in, 90k out) 예시:
  - 낮은 단가: 0.3×0.15 + 0.09×0.60 ≈ **$0.05 + $0.05 ≈ $0.10**
  - 높은 단가: 0.3×0.30 + 0.09×2.50 ≈ **$0.09 + $0.23 ≈ $0.32**
- **한 번 돌릴 때마다 대략 $0.10 ~ $0.35 (약 130원 ~ 450원)** 정도로 보면 됨.
- **한 달에 20번** 돌리면: **약 $2 ~ $7 (약 2,600원 ~ 9,100원)**.
- **한 달에 100번**이면: **약 $10 ~ $35 (약 1.3만원 ~ 4.5만원)**.

→ 실험·소규모 운영 수준이면 **토큰값 감당 가능**한 구간이고,  
  회사 수(`max_companies`)를 5로 줄이면 1회 비용이 더 줄어듦.

## 정확한 요금 확인 방법

- [Google AI for Developers – Pricing](https://ai.google.dev/pricing) 에서 **Gemini 2.5 Flash** 항목 확인.
- 사용 중인 **정확한 모델 ID**(예: `gemini-2.5-flash`, `gemini-2.5-flash-preview-05-20` 등)에 맞는 **Input/Output 단가**를 보고,  
  `(입력 토큰 / 1,000,000) × Input 단가 + (출력 토큰 / 1,000,000) × Output 단가` 로 계산하면 됨.

---

## 비용 줄이는 팁

1. **`max_companies` 줄이기**  
   예: 10 → 5 → 3. 리드·감사·제안 등이 짧아져서 **1회 토큰·비용이 약 30~50% 감소**할 수 있음.
2. **실험 시 회사 수 3~5로 제한**  
   영어/해외 실험은 “구조 검증”이 목적이면 3~5개만 돌려도 충분.
3. **캐시 사용**  
   같은 프롬프트/컨텍스트를 재사용하는 구간이 있으면, API에서 **cached input** 요금이 적용되는 모델/옵션을 쓰면 추가 절감 가능(문서 확인).
4. **모델 다운그레이드**  
   더 저렴한 Flash tier가 있으면(예: gemini-2.0-flash), 품질이 허용하는 범위에서 바꿔서 비용 절감 가능.

## 요약

- **1회 실행**: 약 **$0.10 ~ $0.35** (130원 ~ 450원 수준).
- **한 달 20회**: 약 **$2 ~ $7** (2,600원 ~ 9,100원 수준).
- **1번(한국 유지 + 영어/해외 실험)** 은 회사 수만 3~5로 잡고 돌려도 토큰값 **감당 가능**한 범위이고,  
  정확한 숫자는 위 사이트에서 모델 단가만 확인해 넣으면 됨.
