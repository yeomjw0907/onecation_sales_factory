# Notion AI에게 페이지 만들어 달라고 할 때 쓰는 프롬프트

아래 내용을 복사해서 Notion AI(또는 Notion에서 페이지 만들 때)에 붙여 넣어 사용하세요.

---

## 프롬프트 (복사용)

```
B2B 영업 파이프라인용 데이터베이스 페이지 하나 만들어줘.

요구사항:
1. **전체 페이지 테이블 데이터베이스**로 만들어줘.
2. 데이터베이스 제목: "Sales Pipeline" 또는 "리드 로그" (원하는 걸로).
3. **제목 속성(첫 번째 컬럼) 이름은 반드시 "Name"**으로 해줘. 이 컬럼에는 회사명이 들어갈 거야. (외부 API가 이 이름으로 넣어서 바꾸면 안 됨)
4. 아래 속성들도 추가해줘:
   - **상태** (Select 타입): 옵션은 "리드 발굴", "웹사이트 검토", "랜딩/제안 작성", "이메일 발송", "완료" 정도로.
   - **담당자** (Person 타입)
   - **산업** (Text 타입)
   - **연락처** (Text 타입)
   - **비고** (Text 타입)
5. 각 행(페이지)을 열면 본문에 "[단계명] 요약" 형태로 로그가 자동으로 채워질 예정이야. 그걸 위한 구조만 맞춰줘.
6. 데이터베이스 설명(상단)에 짧게: "Crew AI 영업 파이프라인 로그. Name 컬럼에 회사명, 본문에 단계별 요약이 기록됩니다."

한국어로 속성 이름 써도 되고, 위에 적은 대로 "Name"만 영어로 유지해줘.
```

---

## 짧게 쓰고 싶을 때 (최소 버전)

```
전체 페이지 테이블 데이터베이스 만들어줘. 이름은 "리드 로그".
- 첫 번째 컬럼(제목) 이름은 반드시 **Name** 으로.
- 그다음 컬럼: 상태(Select), 담당자(Person), 비고(Text) 추가해줘.
```

---

## 참고

- **Name** 을 바꾸면 API 연동 시 `.env` 에 `NOTION_TITLE_PROPERTY=실제제목이름` 을 넣어야 함.
- DB 만든 뒤 **Connections** 에서 Notion 통합(Sales Factory 등)을 연결해야 API로 데이터가 들어감.
